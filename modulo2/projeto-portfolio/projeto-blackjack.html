Projeto Blackjack

O projeto Blackjack teve como objetivo a criação do jogo Blackjack (ou em português "21") usando apenas o JavaScript e o HTML. Toda a lógica foi feita em Javascript, utilizando o HTML apenas para a interação do usuário no navegador. Seguem algumas imagens abaixo de como é o funcionamento do jogo e, caso tenha interesse, siga este link do GitHub para dar uma olhada no código!
